# Spring Security + JWT 示例项目
# 说明
本项目是一个示例项目，旨在展示如何在一个 Spring Boot 项目中整合
Spring Security 和 JWT（JSON Web Token）来保护 API。
# 教程
本项目对应的教程可以访问 [Spring Security 4：用户/密码认证](https://blog.icexmoon.cn/archives/866.html)
# 环境
- JDK 21
- Spring Boot 4.0.1
- Mysql
- Redis
- Spring Security
- JJWT 0.13.0
- Mybatis Plus
# 数据库初始化
初始化数据库的 SQL 位于 [sql](https://github.com/icexmoon/learn-spring-security/tree/main/ch4/jwt/src/main/resources/sql)
，其中包含示例数据。用户密码原文均为`password`。
